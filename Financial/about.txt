🎯 Project Title
Financial Risk Modeling: Predicting Loan Default Risk


🧠 Core Problem Statement
Financial institutions must decide which loan applicants are likely to repay and which may default. Approving high-risk borrowers increases losses, while rejecting low-risk borrowers reduces potential profit. This project aims to analyze borrower financial data to:
1. Identify key drivers of loan default.
2. Develop a simple risk scoring framework.
3. Classify applicants into Low, Medium, and High Risk categories.
4. Support data-driven loan approval decisions.
5. Simulate how financial changes (income, loan size, interest rate) impact risk exposure.


#### Project Execution
=> 🔥 Most Predictive Features (Typically)

6. person_emp_length - if the individual has a short employement length
