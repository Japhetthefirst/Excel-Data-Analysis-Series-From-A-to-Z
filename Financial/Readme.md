# 🅱️ F for Financial — Loan-Risk Analysis

## 🏦 Overview

This project is my 6th **A–Z Data Series**, focusing on the financial sector. Using a comprehensive lending dataset of 29,000+ applicants, I built an interactive dashboard in Power BI.
---

## 📊 Objectives

- Who are the applicants?
- How risky are they?
- How do we identify safe applicants?

---

## 🧹 Data Transformation Steps

- Index column in Power Query to enable granular distribution analysis.
- Custom Grouping of applicants by career stage, employment length, age group, income level.
- Data Cleaning............

---

## 📊 Dashboard Visuals

The final **Power BI dashboard** includes:

- 🎯 Visual threshold indicator for portfolio-wide risk averages.
- 📉 Scatter plot showing **Income Scaling by Tenure**.
- 📊 Bar chart segmenting applicants by career levels.
- ⚙️ The Decision Table containing decision logic for determining accpeted or rejected appkicants
- Slicers for Loan Intent (Education, Medical, Venture, etc.)

## 📂 Dataset

**Source:** https://www.kaggle.com/datasets/laotse/credit-risk-dataset?resource=download

---

## 🧠 Key Insights

- Income data is very skewed
- 50.8% of applicants are Home renters
- Higher risk scores strongly correlated with **high exposure rates** (percentage of income committed to loan repayment), regardless of salary.
- Medical loan takers were more likely to default.
